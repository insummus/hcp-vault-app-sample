# Nodejs
## 디렉토리 구조
```
.
├── config/
│   └── default.json     # Vault 접속 정보 및 스케줄링 설정
├── package.json         # Node.js 패키지 정보 및 의존성 목록 (axios, config, node-schedule)
└── vaultClient.js       # 메인 실행 로직 (인증, 갱신, Secret 조회)
```


## 실행 방법
```bash
# 패키지 설치
apt install npm

# 의존성 설치/구성
npm install

# 실행
npm start
```
